# date和datetime

# 转换

* str to datetime

    ```python
    d = datetime.datetime.strptime(str, '%Y%m%d')
    ```

* unix timestamp to datetime

    ```python
    ts = time.time()
    d = datetime.datetime.fromtimestamp(ts)
    ```

* datetime to str

    ```python
    d.strftime('%Y%m%d')
    ```
